**## Testing two biomarker panels for HGSOC diagnosis using random forest models.** 

This repository contains a python code that implements a random forest model to test the ability of two different biomarker panels to diagnose for High Grade Serious Ovarian Cancer. 


**Workspace set up** 

1, Install UV (run in command prompt)
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

2. Move into project directory

3. create virtual environment 
uv init --bare



**Install packages** 

uv add jupyterlab

uv add pandas seaborn matplotlib.pyplot scikit-learn openpyxl



**Open Jupyterlab** 

uv run jupyter lab
